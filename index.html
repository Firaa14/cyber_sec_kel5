Apa itu Hash?

Hash adalah hasil dari algoritma matematika (fungsi hash) yang mengubah data berukuran sembarang menjadi deretan karakter dengan panjang tetap (string). Hash berfungsi sebagai "sidik jari" unik untuk data, memastikan integritas data (tidak diubah), dan sering digunakan untuk mengamankan kata sandi atau dalam blockchain. Hash bersifat satu arah (one-way function), yang artinya tidak bisa dikembalikan ke bentuk aslinya.

Karakteristik MD5:

  • One- way function: tidak bisa dikembalikan ke bentuk asli
  • Deterministik: Input yang sama selalu menghasilkan hash yang sama
  • Avalanche Effect: Perubahan kecil pada input menghasilkan hash yang sangat berbeda
  • Fixed Length: Hasil hash selalu memiliki panjang tetap
  • Collision: dua input berbeda bisa menghasilkan hash yang sama

Tentang MD5

MD5 (Message Digest Algorithm 5) adalah algoritma hash yang menghasilkan nilai hash 128-bit (32 karakter heksadesimal). MD5 dulunya populer untuk verifikasi integritas data, namun sekarang tidak direkomendasikan untuk keamanan karena rentan terhadap collision attacks.

Penggunaan MD5:

  • Checksum untuk verifikasi integritas file
  • Identifikasi unik non-kritis
  • Deduplication data
  • Catatan: Untuk keamanan, gunakan SHA-256 atau SHA-512
